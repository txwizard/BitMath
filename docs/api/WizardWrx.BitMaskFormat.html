ICustomFormatter.Format(String, Object, IFormatProvider)

Format integral types as arrays of bits. Other types are handed off to the system formatting engine.

Declaration
string ICustomFormatter.Format(string format, object arg, IFormatProvider formatProvider)
Parameters
Type Name Description
System.String format

Specify the format string to apply. This method supports a custom format code, B, and the standard G code.
System.Object arg

Specify the integral type to format.
System.IFormatProvider formatProvider

Specify the accompanying format provider.
Returns
Type Description
System.String

If the method succeeds and the format code is B, the return value is a string that represents the integral type as a string of bits, listed from most significant to least.

If the method succeeds and the format code is G, the return value is a hexadecimal representation of the integer, sufficiently padded to hold the maximum value supported by an integer of the specified type.

All other cases are handed over to the default formatting engine and the outcome is undefined, since it depends upon the behavior of a component that is outside of my control.
